# Expense Tracker App

## Description
This is a simple expense tracker app built using Python and Tkinter. It allows users to input items, calculate total expenses, and save data to a file.

## Features
- Add Expense Item with User Input
- Calculate Expense Total
- Save File to Computer

## How to Run
Make sure Python is installed and Run:
python app.py

Or you can hit the play button if you are using Visual Studio
